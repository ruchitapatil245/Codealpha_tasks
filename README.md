# CodeAlpha Full Stack Development Projects

This repository contains projects developed as part of the **CodeAlpha Full Stack Development Program**. These projects demonstrate practical experience in frontend development, backend development, database management, authentication systems, REST APIs, and real-time web applications.

---

## Projects Included

### 1. FlowBoard – Project Management Tool

A full-stack collaborative project management application inspired by tools like Trello and Asana.

#### Features

* User Authentication (JWT)
* Project Creation & Team Collaboration
* Kanban Boards
* Task Assignment & Tracking
* Comments System
* Notifications
* Real-Time Updates using Socket.io

#### Tech Stack

* React 19
* TypeScript
* Tailwind CSS
* Node.js
* Express.js
* Prisma ORM
* SQLite
* Socket.io

---

### 2. ShopVerse – E-Commerce Store

A complete e-commerce platform built with Django.

#### Features

* Product Listings
* Product Search & Filtering
* Product Detail Pages
* Shopping Cart
* User Registration & Login
* Checkout & Order Processing
* Order History
* Django Admin Panel

#### Tech Stack

* Django
* Python
* HTML
* CSS
* JavaScript
* SQLite
